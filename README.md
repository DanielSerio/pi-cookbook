# PI Cookbook

A self-hostable home cookbook application, intended to be installed on raspberry pi.

## Prerequisites

- NodeJS installed
- sqlite3 installed

### Initialize DB

- run: 
    - `yarn create:db` note: if db cli opens, type 'recipes' and close with Ctrl+C
    - `yarn create:dbtables`

### Build

`yarn build`

#### Run

`yarn start`
